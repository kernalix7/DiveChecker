제품 사용 설명서

디지털 압력 감지 학습기 DC-EQ01

01

사용 방법

센서를 활용한 이퀄라이징 연습 순서입니다.

1. 준비

기기 상단의 공기 유입구를 한쪽 콧구멍에 밀착하고, 반대쪽 코를 막습니다.

2. 실습

발살바(Valsalva) 또는 프렌젤(Frenzel) 등 이퀄라이징 기법으로 압력을 형성합니다.

3. 확인

화면의 실시간 그래프로 압력의 강도와 지속 시간을 관찰합니다.

4. 반복

적정 압력 유지와 일정한 파형 형성을 목표로 반복 연습합니다.

Lightning (iPhone 14 이하 / 구형 iPad) 사용자는 별도의 Lightning to USB-C 변환잭이 필요합니다. 옵션 상품으로 구매하실 수 있습니다.

02

앱 화면 안내

DiveChecker 앱의 주요 화면과 기능을 알아봅니다.

홈

디바이스 연결 상태, 실시간 압력값, 캘리브레이션, 정품 인증을 관리합니다.

모니터

30초 슬라이딩 윈도우로 실시간 압력 스트리밍을 확인합니다. 기록 없이 모니터링만 합니다.

측정

시작/정지/일시정지 컨트롤로 세션을 기록합니다. 최대/평균 압력 통계가 실시간 표시됩니다.

기록

저장된 세션 목록을 확인하고, 디바이스별 필터링 및 상세 그래프를 봅니다.

그래프 상세

핀치 줌/드래그로 세션을 탐색합니다. 특정 시점에 메모를 추가할 수 있습니다.

피크 분석

리듬, 압력, 기법, 피로도 4가지 지표로 이퀄라이징 품질을 S~F 등급으로 평가합니다.

03

이퀄라이징 기법 가이드

각 기법의 그래프 패턴을 알면 자신의 이퀄라이징을 정확히 판별할 수 있습니다.

기본

Valsalva

발살바

코를 막고 복부/흉부 압력으로 공기를 밀어내는 기법

그래프 패턴

천천히 올라가는 둥근 커브. 상승/하강이 느리고 피크 폭이 넓습니다.

과도한 힘은 고막 손상 위험. 과압 경고 시 즉시 중단하세요.

중급

Frenzel

프렌젤

혀뿌리를 입천장으로 밀어 올려 공기를 압축하는 기법

그래프 패턴

빠르고 날카로운 스파이크. 상승/하강이 빠르고 피크 폭이 좁습니다.

"K" 또는 "T" 발음 동작으로 연습해 보세요.

고급

Mouthfill

마우스필

성문을 닫고 입안의 공기만으로 이퀄라이징하는 기법

그래프 패턴

초기 높은 피크 후 점차 감소하는 연속 피크. 입안 공기가 줄어들며 압력이 낮아집니다.

고급

Hands-free / BTV

핸즈프리

유스타키오관의 자발적 개방. 코를 막지 않고 이퀄라이징합니다.

그래프 패턴

매우 미세한 압력 변화. 피크가 작지만 일정한 패턴을 보입니다.

04

피크 분석 읽는 법

4가지 지표로 이퀄라이징 품질을 객관적으로 평가합니다.

리듬 점수

피크 간격의 일관성. 변동계수(CV) 기반 — 높을수록 규칙적인 이퀄라이징.

압력 점수

피크 강도의 균일성. 매번 일정한 힘으로 이퀄라이징하는지 평가.

기법 점수

상승/하강 시간, 피크 폭 분석. 날카로운 피크 = 프렌젤에 가까움.

피로도 지수

세션 후반부 압력 감소 추세. 높으면 근육 피로가 누적 중.

종합 등급

S완벽
A우수
B양호
C보통
D미흡
F부족

4가지 지표의 가중 평균으로 산출됩니다. 꾸준한 연습으로 등급을 올려보세요!

05

청소 및 관리

센서의 정확도와 위생을 유지하기 위해 정기적으로 청소해 주세요.

센서 유입구 청소

얇은 면봉 또는 옵션으로 제공되는 전용 청소 도구를 사용하여 공기 유입구의 이물질을 부드럽게 제거합니다.

주의 사항

  • 물이나 액체로 직접 세척하지 마세요
  • 뾰족한 도구로 센서 홀을 찌르지 마세요
  • 에어 스프레이를 직접 분사하지 마세요

보관 방법

사용 후 건조하고 서늘한 곳에 보관해 주세요. 직사광선 및 고온 환경을 피하세요.

전용 청소 도구 및 Lightning 변환잭은 옵션 상품으로 별도 구매하실 수 있습니다.

06

캘리브레이션

현재 대기압을 0 hPa 기준으로 설정하는 과정입니다. 매 세션 시작 전 권장합니다.

캘리브레이션 과정

  1. 센서에서 입/코를 떼고 자연 상태로 둡니다
  2. 앱에서 "캘리브레이션" 버튼을 탭합니다
  3. 3초간 대기압 샘플을 수집합니다
  4. 평균값이 자동으로 기준점(baseline)으로 설정됩니다

고도가 변하거나 장소를 이동한 후에는 반드시 재캘리브레이션하세요. 대기압 변화가 측정값에 영향을 줍니다.

07

펌웨어 업데이트

앱의 설정 탭에서 펌웨어 버전을 확인하고, 최신 버전으로 업데이트할 수 있습니다.

1

현재 버전 확인

설정 → 디바이스 설정에서 현재 펌웨어 버전을 확인합니다.

2

업데이트 시작

"펌웨어 업데이트" 버튼을 탭하면 디바이스가 BOOTSEL 모드로 재부팅됩니다.

3

UF2 파일 복사

PC에서 USB 드라이브로 인식된 DiveChecker에 .uf2 파일을 드래그 앤 드롭합니다.

4

자동 재부팅

업로드가 완료되면 자동으로 재부팅되어 새 펌웨어로 동작합니다.

08

문제 해결

  • 충전 전용 케이블이 아닌 데이터 케이블인지 확인
  • 다른 USB 포트에 연결해 보기
  • 케이블을 분리했다가 다시 연결
  • Android: USB OTG 설정이 켜져 있는지 확인
  • iOS: USB-C iPhone 15 이상 / USB-C iPad만 지원 (Lightning은 변환잭 필요)
  • 캘리브레이션을 다시 진행
  • 센서 홀에 이물질이 없는지 확인
  • 앱을 종료 후 재실행
  • 캘리브레이션을 다시 진행
  • 센서에 수분이 들어가지 않았는지 확인
  • 급격한 온도 변화 후에는 10분 안정화 후 사용
  • 케이블이 제대로 꽂혀 있는지 확인
  • 다른 기기에 연결해서 테스트
  • 문제 지속 시 고객 지원에 문의

안전 및 주의사항

교육 및 실습 전용 — 이퀄라이징 기술 향상을 위한 학습기이며, 그 외의 목적으로 사용할 수 없습니다.

의료기기 아님 — 이관 기능 장애, 중이염 등 질환의 진단/예방/치료 목적으로 사용할 수 없으며, 의료기기로 허가/인증된 제품이 아닙니다.

통증 발생 시 즉시 중단 — 귀 또는 코에 통증이 느껴지면 즉시 사용을 중단하십시오.

방수 아님 — 물/이물질이 유입되지 않도록 주의. 사용 후 건조하고 서늘한 곳에 보관.

제조 품질 안내

  • 제조 과정에서 발생하는 미세한 스크래치나 표면 질감의 차이는 제품 불량이 아닌 공정 특성입니다.
  • 외관에 미세한 표면 차이가 있을 수 있으며, 이는 제품의 기능 및 성능에 영향을 주지 않습니다.
  • 상기 사유로 인한 교환/반품은 불가합니다.
