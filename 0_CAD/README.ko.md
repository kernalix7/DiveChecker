# DiveChecker 하드웨어 설계

프리다이빙 이퀄라이징 압력 측정기의 하드웨어 설계 파일입니다.

## 파일 목록

| 파일 | 설명 |
|------|------|
| `Device_V4.FCStd` | 본체 케이스 설계 v4 (FreeCAD) |
| `BMP280-5V.FCStd` | BMP280 센서 모듈 케이스 (FreeCAD) |

> **참고**: `.legacy-files/` 폴더에는 이전 버전의 설계 파일들이 보관되어 있습니다.

## 소프트웨어 요구사항

- [FreeCAD](https://www.freecad.org/) 0.21+ 권장

## 출력 가이드

### 권장 설정 (FDM)

| 설정 | 값 |
|------|------|
| 레이어 높이 | 0.2mm |
| 인필 | 20% |
| 서포트 | 필요시 |
| 재질 | PLA / PETG |
