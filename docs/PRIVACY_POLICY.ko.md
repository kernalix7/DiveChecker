# 개인정보처리방침

[English](PRIVACY_POLICY.md) | **한국어**

**시행일: 2026년 3월 26일**

## 소개

DiveChecker("본 앱")는 KodeNet("당사")이 개발한 프리다이빙 이퀄라이징 압력 모니터링 애플리케이션입니다. 본 개인정보처리방침은 앱의 정보 처리 방식을 설명합니다.

## 데이터 수집

**본 앱은 어떠한 개인정보도 수집, 전송, 공유하지 않습니다.**

앱에서 생성되는 모든 데이터 — 압력 측정 세션, 차트 메모, 사용자 설정 등 — 는 **기기의 로컬 저장소**에만 저장됩니다(SQLite 데이터베이스 및 SharedPreferences). 외부 서버, 클라우드 서비스 또는 제3자에게 데이터를 전송하지 않습니다.

## 네트워크 접근

본 앱은 인터넷 접근을 **필요로 하지 않습니다.** 릴리즈 빌드에는 `INTERNET` 권한이 포함되지 않으며, 앱은 완전히 오프라인으로 동작합니다.

## 하드웨어 통신

앱은 **USB MIDI** 연결을 통해서만 DiveChecker 하드웨어 장치와 통신합니다. 이 통신은 기기와 연결된 하드웨어 간에 로컬로 수행되며, 어떠한 네트워크를 통해서도 데이터가 전송되지 않습니다.

## 기기 인증

앱은 ECDSA P-256 암호화 서명 검증을 사용하여 연결된 DiveChecker 하드웨어 장치를 인증합니다. 이 과정은 기기 내에서 로컬로 수행되며 외부 서버를 사용하지 않습니다.

## 제3자 서비스

본 앱은 제3자 분석, 광고, 크래시 리포팅 또는 추적 서비스를 **통합하지 않습니다.** 사용자 데이터를 수집하는 제3자 SDK가 없습니다.

## 데이터 저장 및 보관

- **측정 데이터**: 기기의 SQLite 데이터베이스에 로컬 저장
- **사용자 설정**: SharedPreferences를 통해 로컬 저장 (테마, 언어, 알림 설정 등)
- **데이터 내보내기**: 측정 데이터를 JSON 파일로 기기 저장소에 수동 내보내기 가능. 이는 전적으로 사용자가 시작하는 작업이며 기기에 보관됩니다.

기기 설정에서 앱 데이터를 삭제하거나 앱을 제거하면 언제든지 모든 앱 데이터를 삭제할 수 있습니다.

## 아동 개인정보

본 앱은 13세 미만의 아동을 대상으로 하지 않습니다. 전용 하드웨어 장비가 필요한 프리다이빙 훈련용 전문 도구입니다.

## 정책 변경

본 개인정보처리방침은 수시로 업데이트될 수 있습니다. 변경 사항은 시행일 업데이트와 함께 본 문서에 반영됩니다.
