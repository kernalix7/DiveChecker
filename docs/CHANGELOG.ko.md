## [6.0.0] — RTM

### 추가됨
- 크로스플랫폼 디바이스 통신을 위한 USB MIDI SysEx 프로토콜
- 듀얼코어 펌웨어 아키텍처 (Core0: USB/명령, Core1: 160Hz 센서 샘플링)
- OTP 키 저장 지원 ECDSA P-256 디바이스 인증
- 센서 출력 속도 설정 기능 (4-50Hz)
- 펌웨어 IIR 2단계 + 평균화 필터 파이프라인
- PIN 보호 디바이스 설정
- WS2812 RGB LED 상태 표시
- 다국어 지원 (영어, 한국어, 일본어, 중국어)

### 변경됨
- 앱 스택을 MIDI 우선 통신 흐름으로 마이그레이션
- 런타임 안정성 및 재연결 동작 개선
